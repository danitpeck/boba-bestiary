# Boba Bestiary

A monster-manual-style bubble tea tracker built to practice full-stack development with React, Spring Boot, DynamoDB single-table design, and AWS CDK.

## Goals

- Learn DynamoDB single-table design through a playful app.
- Practice Java 17 and Spring Boot API patterns.
- Use React 18 and TypeScript for the frontend.
- Use DynamoDB Local to avoid AWS charges during development.
- Use CDK for infrastructure modeling without deploying at first.
